\section{Overview}
ByteVault is a banking management application with:
\begin{itemize}[nosep]
  \item A \textbf{Next.js} client (frontend).
  \item A \textbf{Bun + Express} API server (backend).
  \item \textbf{Two PostgreSQL 16} databases (Main branch and Sub branch) running in Docker.
  \item Cross-database integration using \textbf{Foreign Data Wrapper (FDW)} and cross-branch settlement correctness using \textbf{Two-Phase Commit (2PC)}.
\end{itemize}
